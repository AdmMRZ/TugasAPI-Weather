# 🌤️ Simple Weather App

Pengguna dapat memasukkan nama kota untuk melihat informasi:
- Suhu (°C)
- Tekanan udara (hPa)
- Kelembapan (%)
- Kecepatan angin (m/s)
- Lokasi (kota dan negara)
